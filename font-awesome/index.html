Restaurant Quiz

Restaurant Quiz is an interactive Android aplication centered around the concept of guessing various restaurant locations across Canada. Developed as a term project for a software design course at the University of British Columbia, it makes use of the GoogleMaps API and Yelp API to provide users with quick, reliable data on current locations and directions.

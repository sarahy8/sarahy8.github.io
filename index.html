Restaurant Quiz is an interactive Android aplication centered around the concept of guessing various restaurant locations across Canada. Developed as a term project for a software design course at the University of British Columbia, it makes use of the Google Maps and Yelp APIs to provide users with quick, reliable data on current locations and directions.

The tech: Android, Java, JUnit, IntelliJ, SVN

EcoGEM

Save gas, the environment and money!
A must-have for the thrifty driver, EcoGEM is a Android application that rewards users for good drving habits. Using the Mojio API, information about the vehicle's fuel efficientcy and distance traveled are obtained, used to calculate a "Gem Score" and if a monthly goal is met, monetary rewards are earned. Developed for nwHacks 2015.

The tech: Mojio API, Android, Java, Android Studio, Git
